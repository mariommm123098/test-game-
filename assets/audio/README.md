# 音频占位目录

当前版本使用 `js/audio.js` 中的 Web Audio API 实时合成雨、低频氛围、火车与提示声，因此离线也能运行。

未来可把正式音频放在这里，建议命名：

- `rain-loop.ogg` / `rain-loop.mp3`
- `rain-canopy-loop.ogg`
- `distant-thunder-01.ogg`
- `train-pass-01.ogg`
- `luggage-wheel-01.ogg`
- `paper-rustle-01.ogg`
- `choice-low-pulse.ogg`
- `summer-campus-loop.ogg`

请使用自制、已授权或明确可商用的素材，并保留来源与授权记录。
